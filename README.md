# hutton-web-mirror

Auto-generated public mirror of the built static output for **dr-hutton** (oralplasticsurgery.com / drlukaspytlik.com).

Purpose: DMCA-resilience and provider diversity. The primary site lives on DigitalOcean; this mirror lives on GitHub Pages (different vendor, different jurisdiction process). If the primary is taken down, DNS can be flipped to GitHub Pages IPs to restore service.

Source repo (private): `davidamoline/hutton-web` — built dist is pushed here on every successful production deploy.

`robots.txt` blocks indexing so this mirror does not compete in SERPs with the primary domain.
